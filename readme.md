# DeadCode
The best theme for Vim and Sublime

## Sublime Text
Not available yet.

## Atom

## Vim
Put `deadcode.vim` file in your `~/.vim/colors/` directory and add the following line to your vimrc file:

    syntax enable
    set number
    colorscheme deadcode
